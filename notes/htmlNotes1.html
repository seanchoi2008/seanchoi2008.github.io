Introduction to HTML

URL

URL is the location of a world wide website

HTML

HTML is the first piece of code indicating that the language is HTML

<!DOCTYPE html>

Doctype is the indicator for which language is being used

Elements

Elements are each of the pieces of a html document

<html>

this element represents the root of an HTML document. All of the HTML document is in it

<head>

The Head is 1 of the pieces of code inside html. Nothig is visible inside

<title>

This is a piece of code that goes at the top of the document. It shows your title for the website

<body>

The body is the other piece of code that goes inside html.The body is the main context

<p>

p is an element that allows you to write things

<h1>

h1 is the biggest header that you can write

<ul>

ul is a piece of code used to make an unordered list

<li>

li is used to make 1 bullet point

<ol>

ol is used to make an ordered list that has nothing in it

Tags

Tags indicate where a element begins and ends.
