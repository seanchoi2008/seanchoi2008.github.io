Images

<img>

Explains an image in html.

Attributes

src

Where you put the URL of an image.

If you want to upload an image, there are 2 ways:

  1. Put the image URL in the src attribute
  2. Save the image and reference the name of the file to the src atribute

height and width

The height and width controls the size of the image.

If you specify one of these, the image will preserve its aspect ratio
